> Validate JSON with a GraphQL schema


## Install

```
$ npm install --save-dev jql-validation
```


## Usage

Define a schema:

```graphql
enum Episode {
    NEWHOPE
    EMPIRE
    JEDI
}

type Character {
  id: ID!
  name: String!
  appearsIn: [Episode]!
  isLegacy: Boolean!
}
```
